# 國立台北大學資訊管理研究所網站前端
2020年改版，使用 ant design vue 建置。

使用 Restful API 與後台串接。

![picture](https://github.com/moriwang/misweb-ant/blob/master/src/assets/logo_ntpu.png)

## 系統架構
* 首頁
* 最新消息
    * 所務公告
    * 招生訊息
    * 學術活動
    * 企業徵才
* 系所簡介
    * 關於本所
    * 教育目標
    * 師資介紹
* 招生資訊
    * 招生簡章
    * 一貫修讀學、碩士學位
    * 台北大學招生資訊網
* 學術研究
    * 學術發表
    * 歷屆論文
    * 電子商務研究
* 學生資訊
    * 課程規劃
    * 修業規定
* 表單與連結
    * 表單下載
    * 畢業校友回填
